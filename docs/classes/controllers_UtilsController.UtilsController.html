  • calculateFromString(val: string, precision?: number): null | number
    • Defined in src/controllers/UtilsController.ts:14

    This method can calculate what's inside a string that represents a calculation (f.e. "1 + 5 - 2" will result in 4)

    Parameters

    • val: string

      the string value that needs to be calculated

    • Optional precision: number

      the precision that the calculation should round to (f.e. if the return value is 5.012 and precision is 2, the endresult should be 5.01) 2 is also the default

    Returns null | number

    The calculated value or null in case that it can't be calculated
